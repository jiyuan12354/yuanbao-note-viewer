​​核心考点提炼​​：

【担保从属性】、【协议管辖】、【独立担保效力】、【违约责任条款】、【仲裁协议效力】

​​三段式结构​​：
  • •
    ​​法理要点​​：
    1. 担保从属性原则：担保合同效力、责任范围、抗辩权均从属于主合同（《担保制度解释》第2、3条）
    2. 管辖确定规则：主从合同约定不一致时，按主合同确定管辖；但单独起诉担保人除外（《担保制度解释》第21条）
    3. 独立担保无效：排除从属性的担保条款无效，但不影响其他部分效力
    4. 违约责任限制：担保责任范围不得超过主债务范围，超出部分无效
    5. 仲裁协议相对性：仅约束协议当事人，不及于未签署方

下栏：配套案例

​​案例事实​​：
张三向李四借款20万元，约定纠纷由A法院管辖。王五提供连带保证，保证合同约定：保证合同独立生效；如王五不能在一周内承担保证责任，需支付违约金10万元；保证合同纠纷由B法院管辖；另约定了C仲裁委仲裁条款。
​​问题句式​​：
对于本案中李四起诉张三与王五所涉及的管辖确定、保证合同效力、违约金约定及仲裁协议效力等问题如何评价？（总分10分）
​​考点提示​​：
​​担保从属性​​、​​协议管辖​​、​​独立担保​​、​​违约责任​​、​​仲裁协议​​
​​对照答案​​：
  1. 1.
    管辖法院为A法院（2分）
    • •
      依据《担保制度解释》第21条，主合同与担保合同管辖约定不一致
    • •
      债权人一并起诉主债务人和担保人的，应当根据主合同确定管辖法院
    • •
      故应由主合同约定的A法院管辖
  2. 2.
    保证合同部分无效（2分）
    • •
      "保证合同独立生效"约定违反担保从属性原则
    • •
      根据《担保制度解释》第2条，独立担保约定无效
    • •
      但不影响保证合同其他部分的效力
  3. 3.
    违约金约定无效（2分）
    • •
      10万元违约金超出主债务20万元范围
    • •
      根据《担保制度解释》第3条，担保责任范围不得超过主债务范围
    • •
      该违约金条款无效，王五仅需在20万元范围内承担责任
  4. 4.
    仲裁协议问题处理（4分）
    • •
      如保证合同约定C仲裁委仲裁：根据或裁或审原则，A法院对保证合同纠纷无管辖权，应裁定不予受理（2分）
    • •
      如借款合同约定D仲裁委仲裁：仲裁协议仅约束签署方，保证合同当事人未达成仲裁协议，D仲裁委无权仲裁保证合同纠纷（2分）
​​不同观点说明​​：
若认为违约金条款是担保人自愿作出的加重责任的承诺，应属有效（该观点与担保从属性原则相冲突，在司法考试中不予采纳）
