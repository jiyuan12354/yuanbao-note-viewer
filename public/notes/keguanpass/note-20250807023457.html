⚖️ ​​口诀记忆​​

​​“代表人选双轨制：人数确定靠推选，人数不定可商定！”​​

​​拆解规则​​：

  1. ​​人数确定的代表人诉讼​​（≥10人）：

    → ​​必须由全体当事人推选​​（法院不可指定）；

  2. ​​人数不确定的代表人诉讼​​（≥10人+权利人未全登记）：

    → ​​先推选​​（当事人自主选）

    → ​​推选不成​​ → ​​法院可与权利人商定​​（《民事诉讼法》第54条）！

✅ ​​法条依据​​（《民事诉讼法》第54条）：

“诉讼标的是同一种类、当事人一方人数众多…​​推选不出代表人的，人民法院可以提出人选与当事人协商​​。”

📚 ​​生动比喻​​

​​将代表人产生比作“选举双通道”——​​

  • ​​通道A（推选）：​​

    人数确定 → 如 ​​小区业主实名投票​​（全体业主推选业委会）；

  • ​​通道B（商定）：​​

    人数不确定 → 如 ​​广场快闪活动​​（参与者流动）→

    → 若无法自选领头人 → ​​警察（法院）推荐人选协商​​（维持秩序）！
