总标题：侦查阶段检察监督权限边界（《刑诉法》第8条核心考点）

​​✓ 本题错误率85%——正确答案为“错误”！必会口诀&陷阱防御指南↓↓↓​​

⚖️ ​​口诀记忆​​

​​“检察监督三不碰：不指挥、不侦查、不越权！”​​

​​拆解三权分立​​：

  1. ​​监督权​​：参加案件讨论、提出书面建议（《刑诉法》第113条）

  2. ​​禁止权​​：

    → ✗ 不得直接指挥侦查

    → ✗ 不得替代实施侦查行为

    → ✗ 不得以“监督”之名干预侦查自主

✅ ​​法条铁则​​（《人民检察院刑事诉讼规则》第561条）：

“人民检察院​​可以派员参加公安机关对重大案件的讨论和其他侦查活动​​，但​​不得干预公安机关的侦查权行使​​。”

📚 ​​生动讲解​​

​​比喻：检察监督是“行车记录仪”——可录不可驾！​​

  • ​​公安机关（驾驶员）​​：掌控方向盘（侦查权），决定路线（侦查方案）；

  • ​​检察院（记录仪）​​：

    → ​​录制路况​​（参加案件讨论）

    → ​​警报提示​​（发纠正违法通知书）

    → ​​禁区​​：​​不能抢方向盘开车​​（直接取证/抓捕）！

    ​​→ 本质​​：“监督”≠“代劳”，越界即程序违法！

“检察引导侦查取证”

仅限​​书面建议​​（非亲手操作）

​​“引导非替代，建议不伸手”​​

​​“此侦非彼侦，场景要分清”​​

🚀 ​​冲刺行动指南​​

  1. ​​【口诀三连】​​

    ​​“监督只能看，不能亲手干；越权必违法，题干伸手马上判！”​​

  2. ​​【情景训练】​​

    • 正确表述：检察院发现违法侦查 → ​​发《纠正违法通知书》​​（书面监督）

    • 错误表述：检察院亲自补充勘验 → ​​越权无效​​

  3. ​​【关联考点】​​
