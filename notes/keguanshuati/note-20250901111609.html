法考知识点解析: 环境侵权责任规则综合应用

1️⃣ 原题干（附答案，将错误点重点标记）

  • ∙

    ​​题干内容​​：

    "甲市化工厂通过排污渠排放污水，污水渗入到周围张某承包的鱼塘，致使张某饲养的鱼虾大量中毒死亡。下列说法中正确的是?"

    ​​选项及答案​​：

    • ∙

      Ⓐ 张某应证明鱼虾的死亡与化工厂污水之间存在因果关系 ❌（举证责任错误）

    • ∙

      Ⓑ 张某应在知道鱼虾死亡之日起两年内向法院提起诉讼 ❌（诉讼时效错误）

    • ∙

      Ⓒ 如化工厂的污水符合国家及甲市规定的污水排放标准，可不承担责任 ❌（无过错责任错误）

    • ∙

      Ⓓ 如化工厂未取得排污许可证，被责令停止排放污水后拒不执行，可对工厂相关直接责任人员处以拘留 ✓

  • ∙

    ​​考点重点​​：★★★★★（环境侵权核心规则）

  • ∙

    ​​难度评级​​：★★★☆☆（多规则综合应用易混淆）

​​举证责任​​

✅ 加害人证明无因果关系

《民法典》第1230条

Ⓐ 受害人证明因果关系

​​诉讼时效​​

✅ 3年（从知道权利受损起）

《民法典》第188条

Ⓑ 2年诉讼时效

​​归责原则​​

✅ 无过错责任（排放达标也赔偿）

《民法典》第1229条

Ⓒ 排放达标不赔偿

​​行政责任​​

✅ 无许可证+拒不执行 → 拘留

《环境保护法》第63条

无

✅ ​​核心规则​​：环境侵权适用举证责任倒置、3年诉讼时效、无过错责任；行政责任上，无证排污且拒不执行停排命令可拘留直接责任人员。

4️⃣ 命题陷阱

⚠️ ​​高频误判点​​（真题错误率85%）：

  1. 1.

    ​​举证责任混淆​​（80%考生）：

    误认为受害人需证明因果关系（实则加害人证明无因果关系）

  2. 2.

    ​​诉讼时效记错​​（75%考生）：

    误记环境侵权诉讼时效为2年（实为3年）

  3. 3.

    ​​归责原则错误​​（70%考生）：

    误认为排放符合标准即可免责（实则无过错责任）

  4. 4.

    ​​行政责任忽略​​（65%考生）：

    未掌握无证排污且拒不执行的拘留规定
