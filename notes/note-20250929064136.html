​​上栏：分析框架​​
​​核心考点提炼​​

【协议管辖的有效性】、【合同履行地的确定】、【法定管辖的适用顺序】、【专属管辖的排除适用】

​​法理要点​​
  • ∙
    协议管辖必须符合《民事诉讼法》第35条规定的形式要件（书面形式）和实质要件（与争议有实际联系的地点、不违反级别管辖和专属管辖）。
  • ∙
    合同履行地的确定是法定管辖的关键。根据司法解释，争议标的为给付货币的，接收货币一方所在地为合同履行地；交付不动产的，不动产所在地为合同履行地；其他标的，履行义务一方所在地为合同履行地。
  • ∙
    管辖权的确定应遵循“协议优先”原则。在没有有效协议管辖或协议不明时，才适用法定管辖（被告住所地或合同履行地）。
  • ∙
    涉及不动产的合同纠纷，并非均属不动产专属管辖（第34条第①项）。只有因不动产的权利确认、分割、相邻关系等引起的物权纠纷才适用专属管辖。常见的房屋买卖合同纠纷、建设工程施工合同纠纷等属于债权纠纷，不适用专属管辖。

​​采分点标注​​
  1. 1.
    审查协议管辖条款是否有效（2分）：形式、内容、是否违反专属/级别管辖。
  2. 2.
    若协议无效或无协议，正确指出法定管辖的连接点（2分）：被告住所地或合同履行地。
  3. 3.
    准确认定合同履行地（3分）：根据争议标的类型，适用司法解释中的具体规则。
  4. 4.
    辨析是否属于不动产专属管辖（2分）：区分物权纠纷与债权纠纷。
  5. 5.
    最终确定有管辖权的法院（1分）：汇总上述分析得出结论。
​​下栏：配套案例​​
​​案例一：协议管辖的有效性争议​​
  • ∙
    ​​案例事实：​​ 住所位于A市的甲公司与住所位于B市的乙公司在C市签订一份设备买卖合同。合同中约定：“如发生纠纷，由合同签订地法院管辖。”后因设备质量问题，甲公司欲起诉乙公司。
  • ∙
    ​​问题句式：​​ 对于本案的管辖法院应如何确定？（10分）
  • ∙
    ​​考点提示：​​ ​​协议管辖的有效要件​​、​​合同签订地​​作为连接点、​​法定管辖​​的补充适用。
  • ∙
    ​​对照答案：​​
    1. 1.
      首先审查协议管辖条款（1分）。本案合同中约定了由“合同签订地法院”管辖，连接点明确，且合同签订地C市与争议有实际联系，该协议管辖条款有效（2分）。因此，本案应由C市人民法院管辖（1分）。
    2. 2.
      若答“协议无效，应适用法定管辖”，得0分，因为本案协议符合要件。
    3. 3.
      若答“由被告乙公司住所地B市或合同履行地法院管辖”，得0分，忽略了有效的协议管辖。
​​案例二：合同履行地及专属管辖的认定​​
  • ∙
    ​​案例事实：​​ 家住D市的王某与E市某房地产公司签订了一份商品房买卖合同，购买位于F市的一处商铺。合同未约定管辖法院。后因逾期交房，王某在D市家中通过手机银行支付了全部尾款后，欲起诉该房地产公司。
  • ∙
    ​​问题句式：​​ 对于王某应向哪个法院提起诉讼的行为如何评价？（10分）
  • ∙
    ​​考点提示：​​ ​​合同履行地的确定​​（争议标的为给付货币）、​​接收货币一方所在地​​、​​不动产纠纷与专属管辖的辨析​​。
  • ∙
    ​​对照答案：​​
    1. 1.
      本案为商品房买卖合同纠纷，属于债权纠纷，​​不适用不动产专属管辖​​（2分）。
    2. 2.
      因无协议管辖，应适用法定管辖，即由被告住所地（E市）或合同履行地人民法院管辖（2分）。
    3. 3.
      确定合同履行地：王某起诉的争议是逾期交房，但其诉讼请求是要求开发商承担违约责任（如支付违约金或赔偿损失），本质上是要求给付货币。因此，​​接收货币一方​​（王某）所在地为合同履行地（3分）。王某的住所地D市即为合同履行地。
    4. 4.
      因此，王某可选择向被告住所地E市人民法院或合同履行地D市人民法院提起诉讼（3分）。
    5. 5.
      若错误认定为不动产专属管辖，则只能由F市法院管辖，扣除第1点和第3、4点相关分数。若将履行地错误认定为“不动产所在地F市”或“履行义务方（开发商）所在地E市”，扣除第3点分数。
